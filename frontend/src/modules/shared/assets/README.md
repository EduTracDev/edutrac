# Shared assets

- **SVGs / images**: Copy reusable SVGs and images from the whitepenguin workspace (e.g. `Heimdall/whitepenguin/src/modules/shared/assets/svgs/`) into `svgs/` or `images/` here as needed.
- **Fonts**: Inter, Source Sans Pro (Source_Sans_3), and Open Sans are loaded in `app/layout.tsx` via Next.js Google Fonts. For local font files (e.g. Avenir from whitepenguin), add them under `public/fonts/` and reference in `globals.css` with `@font-face` if needed.
